# 🎂macaron 마카롱🎂
## 📣프로젝트 소개
SNS가 아닌 두 사람만의 공간에 우리들의 일상을 기록하고 싶다면? 
나와 상대방의 소중한 순간을 기록하고, 일정을 공유할 수 있는 커플 앱 <마카롱>입니다.
## ⚙개발 기간  (2023/07/11 ~ 2023/08/09)
## 💟팀원 및 역할 분담
### 기능

|작업자|작업 내용|
|------|---|
|원경연|회원(로그인/회원가입/커플인증/커플등록), 캘린더(일정 등록/수정)  |
|최영민|마이페이지(프로필/회원탈퇴), 갤러리  |

### 공통 UI
